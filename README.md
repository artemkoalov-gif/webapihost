# webServerGo

Небольшой HTTP‑сервис, который проверяет доступность ссылок, сохраняет результаты в памяти и отдаёт их в JSON или в сгенерированном PDF.

## Требования

- Go 1.25 (или актуальная версия Go)

## Запуск

```bash
PORT=8080 go run ./cmd
```

Сервер поднимается на `http://localhost:$PORT`.

Можно использовать файл `.env` с переменной `PORT=8080`, он подхватывается автоматически.

## API

- `POST /status/v1/` — проверяет ссылки и сохраняет результат.
  - Тело запроса:
    ```json
    {
      "url": ["example.com", "golang.org"]
    }
    ```
  - Ответ: JSON с каждой URL и её статусом, плюс внутренний id списка для последующей выборки.

- `GET /get/v1/` — возвращает выбранные списки в виде PDF.
  - Тело запроса (да, GET с телом):
    ```json
    {
      "links_list": [1, 2]
    }
    ```
  - Ответ: файл `application/pdf` с данными по указанным спискам. Отсутствующие id пропускаются; если ни один не найден, будет 400 с текстом ошибки.

### Примечания

- Все данные о ссылках хранятся только в памяти; перезапуск очищает накопленные списки.
- PDF собирается минимальным встроенным шрифтом Type1 (Helvetica) и содержит каждую запись вида `URL - статус`.
